FAQ

Cloud Hosting — Common Questions

What is cloud hosting and how is it different from VPS?
Cloud hosting distributes your workload across multiple virtual instances on shared physical infrastructure, providing high availability and instant scaling. VPS is a single virtual server with fixed resources. Cloud hosting suits variable or spiky traffic, while VPS is better for consistent, predictable workloads.
How does billing work for cloud hosting?
Cloud hosting plans are billed monthly with no long-term contract. You can cancel at any time without penalties. We charge a flat monthly rate per tier, making costs predictable regardless of actual usage within your plan.
Can I scale my cloud resources without downtime?
Yes. You can scale CPU and RAM up or down instantly through our control panel with zero downtime. Storage scaling typically completes in under 5 minutes with a brief maintenance window.
What is the SLA uptime guarantee for cloud hosting?
All Orbit Cloud Cloud Hosting plans carry a 99.99% uptime SLA. The distributed architecture means your application stays online even if one physical node fails, thanks to automatic live migration to healthy nodes.
Is managed cloud hosting available?
Yes. All cloud plans include basic infrastructure management. For a fully managed experience including OS updates, security hardening, and application-level support, contact our sales team about adding a managed services package.
