FAQ

WordPress Hosting — Common Questions

Do I need technical knowledge to set up WordPress on OrbitHost?
No technical skills are required. WordPress comes pre-installed on all our WordPress hosting plans. If you prefer a fresh install, our 1-click installer in cPanel deploys WordPress in under 60 seconds.
Will my WordPress site be automatically backed up?
Yes. Business and Pro plans include automated daily backups stored offsite for 30 days. Starter plans include weekly backups. You can restore your site to any backup point in a single click from your control panel.
How does WordPress performance compare to regular shared hosting?
Our WordPress hosting uses LiteSpeed web server with full-page caching, up to 5× faster than standard Apache-based shared hosting. Combined with NVMe SSD storage and built-in object caching, most WordPress sites load in under 500ms.
Can I install WordPress plugins and themes freely?
Yes — you have full control over your WordPress installation. You can install any plugin or theme from the WordPress repository or upload custom ones. We do not restrict installations except those that create security vulnerabilities or abuse server resources.
What happens if my WordPress site gets hacked?
Business and Pro plans include automated malware scanning and removal. If malware is detected, our security team will clean your site. Pre-configured security hardening — brute force protection, file integrity monitoring — is active on all plans.
