FAQ

Dedicated Server Questions

How long does server deployment take?
Standard dedicated server configurations are deployed within 24–48 business hours of order confirmation and payment. Custom configurations may require 3–5 business days. We will keep you informed throughout the provisioning process with status updates.
Can I customise the hardware configuration?
Yes. Contact our enterprise sales team for fully customised server configurations. We can accommodate specific CPU models, custom RAM amounts (up to 2TB), mixed storage configurations (SSD + HDD), additional network cards, and GPU installations for HPC or machine learning workloads.
What is included with dedicated server support?
All dedicated server plans include hardware monitoring, datacenter access, power and network redundancy, and basic hardware replacement (failed drives, etc.) at no extra charge. Managed services (OS management, security patching) are available as add-ons starting at $49/month.
