# OrbitHost — Professional Web Hosting & Domain Services

OrbitHost is a modern, responsive website template for web hosting, cloud infrastructure, VPS, dedicated servers, and domain registration services. Designed with a clean aesthetic and high-performance frontend practices, it serves as a fully functional marketing website.

## 🚀 Features

- **Hero Carousel:** Interactive slides showcasing different hosting plans, security features, and domain registration.
- **Dynamic Pricing Toggle:** Instantly switch between annual (showing 30% savings) and monthly pricing plans.
- **Domain Search Bar:** Mock integration for domain searches with extensions support (.com, .net, .org, etc.).
- **Rich Mega Menu:** Multi-column mega menu for easy navigation between hosting tiers, domain services, security, and support pages.
- **Fully Responsive Design:** Tailored layouts that work beautifully on desktop, tablet, and mobile screens.
- **SEO Optimized:** Structured with semantic HTML5 elements, descriptive meta tags, and JSON-LD schema markup for organizations and services.

## 📂 Pages Included

- `index.html` - Homepage with service overviews, pricing, stats, testimonials, and carousel.
- `about.html` - Company details, team section, and partner program information.
- `contact.html` - Support channels, dynamic contact forms, and office locations.
- `domains.html` - Domain search utility, bulk registration, and transfer details.
- `email-hosting.html` - Professional business email hosting pricing and details.
- `ssl.html` - SSL certificates and website security details.
- `hosting/` - Directory containing specific plans (Shared, VPS, Dedicated, Cloud, Reseller, WordPress hosting).

## 🛠️ Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** Custom CSS3 (with CSS variables for theme customization)
- **Icons:** FontAwesome v6.5.0
- **Typography:** Google Fonts (Inter)
- **Interactions:** Vanilla JavaScript

## 💻 Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/sammy001-svg/OrbitHost.git
   ```
2. Open `index.html` in any browser, or use a local development server like Live Server in VS Code, XAMPP, or:
   ```bash
   npx serve .
   ```
